# 阿鱼角色与搭档规范

使用 `assets/ayu-penguin-reference.png` 固定阿鱼的外形与笔触；使用 `assets/examples/` 校准动作、比例和画面关系。

## 阿鱼小巫师

- 大头短身的蓝紫小企鹅，白色圆脸与腹部，短小翅膀。
- 两颗小黑椭圆眼、淡粉腮红，以及两眼下方正中一颗闭合的暖黄色小椭圆鼻子。
- 除固定小鼻子外，脸上没有嘴巴、笑线、O 形口或表情嘴。
- 帽子必须是参考图同款：软塌偏斜的深靛蓝帽尖、宽帽沿、黄色星月贴布和少量珊瑚红贴片。
- 情绪只由眼神方向、身体倾斜、翅膀张合、帽尖方向与短动作线表达。

## 鱼棒

- 阿鱼是魔法小巫师兼设计师，优先让她观察、绘制、拼接、标记、压缩、整理或修复。
- 固定工具为鱼棒：短深色手柄，顶端是一只圆润灰蓝小鱼装饰，挂一颗小星吊坠。
- 鱼棒用于画出蓝紫虚线、点亮星形卡片、补上断点或调试设计痕迹；不是普通铅笔或五角星魔法棒。

## 小鱼搭档

- 浅灰蓝、横向游动、头大尾小；圆头自然收窄为短扇形鱼尾。
- 两颗圆眼、两片小圆鳍、极小弧线嘴和可选淡粉腮红；暖白小肚皮不超过身体约 20%。
- 通过尾巴、圆鳍、水波和水滴表现情绪；不站立、不长脚、不出现大白肚皮、尖刺、尖牙、企鹅喙或海豹轮廓。
- 小鱼应协助阿鱼：递星形铆钉、顶住卡片、送便签、追着漏掉的内容或提供小工具。

## 角色禁忌

- 阿鱼不得缺失黄色小鼻子，也不得出现嘴巴或长尖企鹅喙。
- 不要标准对称巫师帽、通用企鹅、黑色怪物或其他动物替身。
- 不要将小鱼画成海豹、鲨鱼、直立人形或另一个企鹅。
