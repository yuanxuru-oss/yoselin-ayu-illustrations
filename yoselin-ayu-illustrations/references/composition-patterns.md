# 阿鱼魔法设计构图

每图只选一种认知动作。构图应像阿鱼正在白纸上把一个问题设计出来并修好。

## 可复用场景

| 认知动作 | 画面隐喻 | 阿鱼动作 |
| --- | --- | --- |
| 分类 | 月亮抽屉或三格收纳盘 | 鱼棒给卡片盖星形标记 |
| 复用 | 一张大草图变三张小卡 | 用鱼棒与设计尺折叠、展开 |
| 承接 | 两个平台之间的点线桥 | 鱼棒补点，小鱼递星形铆钉 |
| 收集 | 低矮的圆形灵感井 | 用鱼棒钩起便签，小鱼推回漏掉的卡 |
| 压缩 | 简化的小型压机 | 压下杂乱想法，推出一张清晰卡 |
| 养成 | 一只透明灵感罐 | 用虚线调试、微调卡片与星屑 |
| 建信任 | 一块块卡片搭成桥 | 小心放下证据卡，不跨越空格 |

## 动作规则

- 阿鱼不能正面站立举物；使用单脚、前倾、后仰、蹲下、跨步或伸翅等不对称姿势。
- 帽尖、鱼棒、翅膀与虚线应指向同一个视觉方向。
- 加一处“正在设计”的痕迹：半擦的构图框、试画虚线、倾斜便签或未完成的桥段。
- 小鱼每次都做一件小但明确的协作动作，避免只是漂浮装饰。
